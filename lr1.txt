Права доступ к любому объекту устанавливается администратора при помощи команды chmod 
1 Вариант:
sudo chmod 775 /home/tmp3
Установили полный доступ к католону /home/tmp3 для владельца каталога и группы владельца а для всех остальных пользователей право на просмотр (х) и чтение каталога (г) 
2 Вариант:
sudo chmod o+w /home/tmp3
Установили право лдя всех пользователей на запись в катаог /home/tmp3
sudo chmod g-w /home/tmp3
убрали право на запись у группы владельца каталога /home/tmp3

Владельца и группу владельца любоо объекта администратора можно поменять при помощи команды chmod 
sudo chmod tmp4:student /home/tmp3/F1
Изменили владельца каталога /home/tmp3/F1 на пользователя tmp4 а группу владельца на группу student
sudo chown :tmp4 /home/tmp3/readme
Изменили гркппк владельцев файла /home/tmp3/readme на группу tmp4

Для работы в других существующих группах можно испротзовать команду newqr
newqr tmp5
Для смены группы необходимо знать пароль группы 
после смены группы можно получить общий доступ к объектам по группе владельцев каждо объекта
После завершения операций под общей группой выход из группы в свою группу по умолчанию осущствляется командой exit
